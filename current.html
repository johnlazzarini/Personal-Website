Current Projects

Faux Clothing/Retail Application

I've always admired the clean web application designs opertated by modern clothing companies -- Zara, H&M, Abercrombie & Fitch, and many others all have websites that entice visitors with beautiful and intuitive UI.

I think that making a web application in this style would be a great way to flex a variety of web dev muscles -- databases, shopping carts, responsive design. This app intends to demonstrate exactly these sorts of features, but is very much a work in progress.

United States Capitals Web Application (Not Yet Live)

I'm currently working on a web application consisting of a map of the United States -- users can hover over various states, which prompts the application to fetch and display information about the capital, main exports, landmarks or significant features, and other notable qualities. This is my first attempt at building a fully featured web application, and is my own personal project. This page will soon link to a working sample of what I've been able to do so far.
